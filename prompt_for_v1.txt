VERSION 1 — INDUSTRIAL (Endurance Employees Only):
  - App is locked behind an ACCESS CODE on first launch
  - Endurance will distribute 20–30 unique codes to employees
  - On first install (App Store / Play Store / direct APK), the app shows

  - Employee pastes/types the correct code → app unlocks permanently for

  - Wrong code → error message, app stays locked

  - Once unlocked, full app is available with no further code prompts
  - Locked screen has: Endurance logo, "Enter Access Code" field,
    "Activate" button, support contact info

VERSION 2 — PUBLIC VERSION (Future, Not Built Now):
  Anyone installs and uses without any code. Noted for future reference only.
